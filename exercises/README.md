# Exercises Overview

This folder contains all guided exercises for the Spec-Driven Product Development course. Each exercise is built around the Relay scenario -- you're a PM at a Series B project management SaaS company, working on an AI-powered "Smart Dependencies" feature.

Every exercise has two parts:
1. **Template Exercise** -- Apply the lesson to the Relay scenario using the provided templates
2. **BYOP (Bring Your Own Product)** -- Apply the same lesson to a real feature or product from your own work

## Tips

- **Work in order within a level.** Each exercise builds on the previous one.
- **Commit after each exercise.** Your git history is part of your learning artifact.
- **Don't skip BYOP.** The Relay exercises build your understanding. The BYOP exercises build your skill.
